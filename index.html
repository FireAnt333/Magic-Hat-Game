The Magic Hat - V2.3.1

The wizard has lost his magic hat in a field full of holes at night! Help him find it safely!

Controls

W/A/S/D

Move up, left, down and right

F

Find a solution path to the hat

G

Request new game from the menu, or

return to game after viewing solution

M

Return to the menu from the game

Y/N

Answer "yes" or "no" to a prompt

(*Not used in current build)
